Descrição
Site estático que consome a API pública PokéAPI (https://pokeapi.co/) para exibir cards dinâmicos dos Pokémon (nome, imagem, tipos).

Como testar localmente
1. Extraia os arquivos.
2. Abra o projeto no VS Code e use Live Server OU simplesmente rode um servidor local (por exemplo, `python -m http.server`).
3. Abra `index.html` via http://localhost:5500 (ou porta do Live Server).
4. Use o campo de busca (digite um nome e pressione Enter) e clique em 'Carregar mais' para ver mais Pokémons.

API utilizada
- Endpoint base: https://pokeapi.co/api/v2/pokemon
- A API é pública e não requer autenticação. Retorna objetos com sprites (imagens), tipos e demais atributos.
